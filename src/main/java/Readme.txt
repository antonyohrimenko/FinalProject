Реализация классов

Presentation Layer
- Main.java: Основной класс, запускающий приложение.
- ConsoleUI.java: Обеспечивает консольное взаимодействие с пользователем, вызывая методы из бизнес-слоя.

Business Layer
- AuthService.java: Класс, который отвечает за авторизацию пользователей.
- RegistrationService.java: Класс, который отвечает за регистрацию новых пользователей, включая проверку уникальности имени пользователя и валидацию пароля.
- PasswordEncryptor.java: Класс для шифрования паролей. Используем, например, SHA-256 для хеширования паролей.

Data Access Layer
- UserRepository.java: Класс, который хранит и управляет данными пользователей (в памяти или в файле).
- User.java: Модельный класс, представляющий пользователя с полями для имени, пароля и других необходимых данных.
